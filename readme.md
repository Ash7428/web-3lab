# Лабораторная работа 3 - Игра 2048 

Лабораторная работа по дисциплине **«Веб-программирование»**.  
Реализация классической игры **2048** с сохранением состояния, таблицей лидеров и адаптивным управлением.

---
Веб-версия игры **2048**, в которой пользователь объединяет плитки одинакового номинала, стремясь получить значение **2048** или установить максимальный рекорд.

Игра реализована **без использования CSS- и JS-фреймворков** 

---

## Реализованный функционал

### Игровая логика
- Динамическое создание игрового поля **4×4**
- Перемещение плиток во всех направлениях (вверх, вниз, влево, вправо)
- Корректное слияние плиток одинаковых значений
- Начальное появление **2–3 плиток** со значениями `2` или `4`
- После каждого успешного хода появляется **1–2 новые плитки**
- Подсчёт очков как суммы значений объединённых плиток
- Проверка окончания игры (отсутствие возможных ходов)

### Управление состоянием
- Кнопка **New Game** — начать игру заново без перезагрузки страницы
- Кнопка **Undo** — отмена одного последнего хода
- Сохранение текущего состояния игры в `localStorage`
- Возможность продолжить игру после обновления страницы

### Таблица лидеров
- Таблица **Top-10 рекордов**
- Поля: *Имя игрока, Очки, Дата сохранения*
- Сохранение рекордов в `localStorage`
- Модальное окно с возможностью очистки таблицы

### Адаптивность и управление
- Управление **стрелками клавиатуры** на десктопе
- Управление **кнопками и свайпами** на мобильных устройствах
- Элементы управления отображаются только во время активной игры
- Корректное отображение на ноутбуках, планшетах и смартфонах

### Визуальное оформление
- Уникальный цвет для каждого значения плитки
- Плавные анимации перемещения и слияния плиток
- Модальные окна для Game Over и таблицы лидеров
